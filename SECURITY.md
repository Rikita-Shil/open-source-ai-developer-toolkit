# Security Policy

## Reporting a Vulnerability

If you discover a security issue in this project, please report it by opening a private security advisory or creating an issue with sufficient details to reproduce the problem.

Please avoid publicly disclosing vulnerabilities until they have been reviewed.

## Supported Versions

| Version | Supported |
|---------|-----------|
| 1.x | ✅ |
| < 1.0 | ❌ |

## Scope

This repository contains developer productivity skills and documentation. Security improvements and responsible disclosure are always welcome.
