# Installation

This guide explains how to install and test the Open Source AI Developer Toolkit locally with Claude Code.

## Requirements

Before starting, make sure you have:

- Git installed
- Claude Code installed
- A valid Claude Code account
- Access to a terminal
- A code editor such as Visual Studio Code

## Clone the Repository

```bash
git clone https://github.com/Rikita-Shil/open-source-ai-developer-toolkit.git
cd open-source-ai-developer-toolkit
