# Code of Conduct

## Our Pledge

We are committed to providing a welcoming, respectful, and inclusive environment for everyone.

## Expected Behaviour

- Be respectful.
- Be constructive.
- Welcome newcomers.
- Give helpful feedback.

## Unacceptable Behaviour

- Harassment
- Personal attacks
- Discrimination
- Spam

## Enforcement

Project maintainers may remove contributions that violate this code of conduct.
