# Contributing

Thank you for your interest in contributing to the Open Source AI Developer Toolkit.

## How to contribute

- Fork the repository.
- Create a new feature branch.
- Keep commits focused on one logical change.
- Follow the existing folder structure.
- Update documentation when adding new skills.
- Test your changes before submitting a pull request.

## Commit style

Use short, descriptive commit messages.

Examples:

- Add Docker reviewer skill
- Improve README installation guide
- Fix typo in API reviewer documentation

## Reporting issues

Please include:

- Description of the problem
- Expected behaviour
- Actual behaviour
- Steps to reproduce
