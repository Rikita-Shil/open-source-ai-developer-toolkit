# API Reviewer

## Overview

API Reviewer helps developers evaluate RESTful and HTTP APIs for correctness, security, maintainability, and developer experience.

It reviews API implementations against common backend engineering practices and provides practical recommendations before deployment.

## What it Reviews

- REST API design
- HTTP methods
- Status codes
- Authentication and authorization
- Request validation
- Error handling
- API versioning
- Documentation quality
- Security considerations
- Performance considerations

## Typical Use Cases

- Reviewing Express.js APIs
- Reviewing FastAPI services
- Reviewing Spring Boot APIs
- Reviewing ASP.NET APIs
- Reviewing Node.js backend projects

## Expected Output

The reviewer should provide:

- Overall API quality score
- Major issues
- Minor improvements
- Security observations
- Best-practice recommendations
- Final deployment readiness assessment
